# todolist

## 运行环境
- 服务器：Node.js
- 数据库：MySQL


## 项目说明
- 框架：Express
- 项目数据库：database文件夹中的web_todolist.sql文件
- 前端：Bootstrap、jQuery、Ajax和moment.js（一个日期和时间转换的js库）
- 后端：Node.js、Express、RESTful接口设计
